##Student Gradebook 🎓

A desktop application built with Java Swing that allows teachers to manage and track student grades.

## Features
- Add students with Name, ID, Subject and Grade
- Automatically calculates letter grade (A, B, C, D, F)
- Calculates and displays class average
- Delete individual rows
- Clear all data
- Input validation for all fields

## How to Run
Make sure you have Java installed, then:
javac Gradebook.java
java Gradebook

## Built With
- Java
- Java Swing
